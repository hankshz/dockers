# Hbase cluster

An example for Apache Hbase in cluster mode

  - Build & Install Hbase
  - Run sample script

Usage

  - Build the docker image of Hadoop & HBase:
    ```sh
    ./script/build-image.sh
    ```
  - Create the cluster:
    ```sh
    ./script/start-instance.sh
    ```
  - Run the example:
    ```sh
    ./script/run-example.sh
    ```
