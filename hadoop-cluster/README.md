# Hadoop cluster

An example for Apache Hadoop in cluster mode

  - Build & Install Hadoop
  - Run map reduce example

Usage

  - Build the docker image of Hadoop:
    ```sh
    ./script/build-image.sh
    ```
  - Create the cluster:
    ```sh
    ./script/start-instance.sh
    ```
  - Run the example:
    ```sh
    ./script/run-example.sh
    ```
